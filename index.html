Зона комфорту - модерний готель у серці Харкова


Ми пропонуємо:

  • комфортне розміщення у зручні для Вас дати
  • номери стандарт та люкс
  • номери для роботи і для відпочинку
  • місце для паркування
  • можливість проживання з дітьми
  • можливість замовлення харчування у ресторані готелю
  • готель для тварин на випадок, якщо Ви подорожуєте з улюбленцем
  • цілодобовий доступ до бомбосховища

Хочеш до Зони комфорту? Бронюй зараз!
